# AgenTravel

Agente autonomo de recomendaciones turisticas con pagos x402.

Alguien pregunta que hacer en una ciudad en fechas especificas, el agente responde cobrando via x402.

## Estructura

```
AgenTravel/
├── knowledge/      # Reglas del concurso, requisitos, criterios de evaluacion
├── prompts/        # Prompts del agente (consultas de viaje, respuestas)
├── src/            # Codigo fuente del agente
├── tools/          # Documentacion de herramientas y APIs usadas
└── docs/           # Notas, ideas, arquitectura
```
